Reglas de negocio:
•	El sistema, deberá tener tres tipos de usuarios: Cliente, Administrador, y SuperAdministrador.  
•	Todos los usuarios deberán poder autentificarse y autorizarse en el sistema permitiendo a los clientes registrase en caso de no tener cuenta. 
•	Todos los usuarios, podrán visualizar y filtrar los productos por su tipo, solamente si iniciaron sesión previamente. 
•	El Administrador deberá poder cambiar su nombre de usuario.
•	El cliente deberá poder cambiar su contraseña.
•	El cliente podrá comprar uno o varios productos.
•	El cliente podrá consultar todas las reviews de un producto ingresando el id del mismo.
•	Las compras, deben tener un cliente, y uno o varios detalles de ventas en el que se especifiquen el producto y la cantidad del mismo. 
•	Los clientes deberán poder consultar su historial de compras, visualizando los detalles del producto que compró junto con su cantidad.
•	Un cliente previamente logueado, podrá hacer una review de un producto el cual haya comprado.
•	Los productos, podrán tener cero o varias reviews. 
•	Las reviews deben estar directamente relacionadas con un producto específico y un cliente. 
•	Las reviews deben tener un contenido. 
•	Tanto el Administrador como el SuperAdministrador, deberán tener la posibilidad de:
	-	Añadir productos. 
	-	Eliminar productos. 
	-	Modificar productos. 
	-	Consultar productos. 
•	El SuperAdministrador, deberá poder:
	-	Añadir usuarios. 
	-	Eliminar Usuarios.
